# GeMS-ArcPro-Toolbox


## Features

* Strat Name Look Up
* MapUnitPoly Neighbors- Checking adjacent polygons for duplication
* Transfer Data(NCGMP09 to GeMS)
* GeMS validation for Arcgis Pro

ArcGIS Desktop
